# Installation
### Clone this repo onto your local machine
  - WINDOWS:
    - Clone into %USER_HOME%/AppData/Local/nvim
    
### Install Neovim
- https://github.com/neovim/neovim/blob/master/INSTALL.md
- For Windows:
  - Install Visual Studio build tools for the C compiler
  - Install chocolatey: Run choco install mingw in admin powershell.
  - Run refreshenv to refresh to envoriment.

### Start Neovim
- Navigate to the directory where you want to open to in Neovim
- Use the command 'nvim .'
